# Bike Sales Dashboard (Excel)

## 📊 Project Overview
This project analyzes a Bike Sales dataset using Microsoft Excel to uncover customer purchasing patterns and build an interactive dashboard.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## 🔍 Key Analysis
- Average income per purchase (Gender-based)
- Customer age brackets
- Commute distance impact
- Region and education filters

## 📈 Key Insights
- The Middle-Age bracket is the only segment where buyers outnumber non-buyers, accounting for nearly 80% of all bike purchases in the dataset.
- Proximity is the primary driver of intent; customers with a commute of under 1 mile represent the largest surge in sales, with a sharp decline as distance increases.
- While 0-1 car households are prime targets, there is a massive drop-off in purchase intent once a household owns 2 or more vehicles.
